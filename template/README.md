## Customization

The theme uses CSS variables that you can override:

```css
.theme-my-custom-theme {
  --theme-accent: #6366f1;
  --bg-primary: #0a0a1a;
  /* ... */
}
```

## Development

To customize this theme:

1. Fork this repository
2. Edit `theme.json` for theme metadata
3. Edit `theme.css` for styling
4. Update the version in `theme.json`
5. Create a new release
